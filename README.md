# Bus Booking Management System

A web-based bus booking management system built with PHP and HTML.

## Features

- Bus booking functionality
- Database integration
- User-friendly interface

## Project Structure

```
bus_booking/
├── database/
│   ├── book_bus.php.txt
│   ├── db.php.txt
│   └── get_buses.php.txt
└── index.html.txt
```

## Setup

1. Clone the repository
2. Set up XAMPP server
3. Place files in `htdocs/bus_booking/`
4. Configure database connection in `database/db.php`
5. Access via `localhost/bus_booking/`

## Requirements

- XAMPP/WAMP server
- PHP 7.0+
- MySQL database

## Installation

1. Start Apache and MySQL in XAMPP
2. Create database for the application
3. Import database schema (if available)
4. Update database credentials in configuration files

## Usage

Navigate to `http://localhost/bus_booking/` in your web browser to access the application.
